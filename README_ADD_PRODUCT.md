# Как добавить товар в магазин

Чтобы добавить новый товар, не нужно программировать. Достаточно создать новый текстовый файл в папке `/content/products/`.

1. **Создайте файл**: Скопируйте существующий файл (например, `classic_candle.md`) и переименуйте его (например, в `new_candle.md`).
2. **Отредактируйте текст**: Откройте его в редакторе GitHub. В верхней части вы увидите блок между `---`. Измените там данные:
   ```markdown
   ---
   title: "Название товара"
   price: 1000
   description: "Краткое описание"
   has_scent: true
   has_color: true
   image: "covers/cover_classic.png"
   ---

   Здесь напишите подробное описание товара. Можно использовать обычный текст.
   ```
3. **Сохраните**: Нажмите кнопку "Commit changes" в GitHub.

После этого сайт автоматически пересоберется в течение пары минут, и товар появится в каталоге.
