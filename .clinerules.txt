Стек: Используй строго один файл index.html. Tailwind CSS через CDN.

Контент и Цены: Данные бери ТОЛЬКО из моих файлов и CSV-прайса. Никакой «рыбы» и выдуманных цен.

Изображения (КРИТИЧНО): Используй относительные пути. ЗАПРЕЩЕНО менять расширения и названия файлов. Перед написанием кода проверь фактическое наличие файла в папках Мини/, Мэлтсы и саше/, Пасхальная коллекция/ и Обложки/.

Карточка товара: Все карточки (включая «Спираль» и «Заяц») должны иметь одинаковую структуру: фиксированные пропорции фото aspect-[4/5], object-cover и единую сетку grid.

Адаптивность: Mobile-first подход. Сайт должен идеально выглядеть на iPhone.

Чистота и Лимиты: Минимум комментариев. Код должен быть компактным. Не удаляй существующий рабочий функционал формы обратной связи при добавлении новых секций.

Форма связи: Обратная связь — через форму на странице контактов (отправка на email). 
Для всех вызовов API используй параметр max_tokens: 4000. Это критическое ограничение.
