# PHICANDLES - Интернет-магазин ароматических свечей ручной работы

Этот репозиторий содержит исходный код для веб-сайта бренда PHICANDLES, специализирующегося на производстве ароматических свечей ручной работы.

## О проекте

PHICANDLES - это бренд ароматических свечей ручной работы, созданных с любовью и вниманием к каждой детали. Мы используем только натуральные материалы высочайшего качества, чтобы создавать уникальные изделия, которые наполнят ваш дом теплом, уютом и изысканным ароматом.

## Технологии

Сайт разработан с использованием следующих технологий:

- HTML5
- CSS3
- JavaScript (vanilla)
- Tailwind CSS (через CDN)
- Google Fonts (Playfair Display)

## Структура проекта

Сайт имеет следующую структуру:

- Главная страница с информацией о бренде и коллекциях
- Отдельные разделы для каждой коллекции:
  - Цветы
  - Классик
  - Декор
  - Мини
  - Ароматы для дома
  - Пасхальная коллекция
- Корзина для оформления заказов
- Контактная информация

## Функциональность

- Адаптивный дизайн для мобильных и десктопных устройств
- Одностраничное приложение (SPA) с навигацией по разделам
- Корзина с возможностью добавления товаров, изменения количества и удаления
- Форма оформления заказа
- Форма обратной связи

## Запуск проекта

Для запуск проекта локально достаточно открыть файл `index.html` в любом современном браузере.

## Размещение на GitHub Pages

Для размещения сайта на GitHub Pages выполните следующие шаги:

1. Убедитесь, что у вас есть аккаунт на GitHub.
2. Создайте новый репозиторий или используйте существующий.
3. Загрузите все файлы проекта в репозиторий:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/username/PHICANDLES_WEB.git
   git push -u origin main
   ```
4. Перейдите в настройки репозитория (Settings).
5. Прокрутите вниз до раздела "GitHub Pages".
6. В разделе "Source" выберите ветку "main" и папку "/ (root)" и нажмите "Save".
7. Через несколько минут ваш сайт будет доступен по адресу: `https://username.github.io/PHICANDLES_WEB/`

Примечание: Замените "username" на ваше имя пользователя GitHub.
